# Jenisha Patel — Portfolio

A personal portfolio site built as a standalone React + Vite application.

## Stack
React 18 · React Router 6 · Vite · Tailwind CSS · shadcn/ui · Framer Motion

## Getting started

```bash
npm install
npm run dev       # local dev server
npm run build     # production build -> dist/
npm run preview   # preview the production build locally
```

## Editing content

All page content (projects, blog posts, experience, leadership, skills,
certifications, awards, photos, education, etc.) lives in `src/data/` as
plain JS — no CMS, no backend, no database. See `src/data/README.md` for
the full guide to adding, editing, and organizing content.

## Deploying

This builds to a static `dist/` folder like any other Vite app — deploy it
to Netlify, Vercel, GitHub Pages, Cloudflare Pages, or any static host.
